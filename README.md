Neu
===

Neu is an auto-differention library to construct, train and test deep neural networks written in C++14.

This software is released under the MIT License, see LICENSE.

Dependency
---

- OpenCL
- Boost
- Boost.Compute
- yaml-cpp
- freeimageplus
- cldoc(optional)

Features
---

**Layer**
- inner production
- spacial convolution
- max pooling
- average pooling
- dropout

**Activation**
- ReLU
- leaky ReLU
- sigmoid
- tanh

**Optimizer**
- fixed learning rate
- momentum

Installation
---

```
mkdir build && cd build
cmake ..
make
```
